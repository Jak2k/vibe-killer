# Vibe Killer

Welcome! Thanks for stopping by and happy hacking.

## Testing

Check the `examples` folder to find projects that we tested while developing this project. The `existing-hostile` folder contains sub-repos with projects that already have adversarial AGENTS.md files.

## Mission Statement

Consent is important. In recent years, Free and Open Source projects have been increasingly burdened, with mountains of slop commits, causing contributor burnout.
Regardless of what _you_ may think, [there are many valid reasons for people to hate AI](https://www.youtube.com/watch?v=ERiXDhLHxmo).

Because of this, we have made it our goal to try and combat AI contributions by sabotaging the quality of slop-generated code and making the use of programming tools more costly and tedious.
